OUR GOAL


THE GOAL OF THIS PROJECT IS TO PROVIDE A SIMPLE, INTERACTIVE WAY TO VISUALISE THE IMPACT OF COVID-19 ON INDIA AND THE WORLD - FOR MONITORING THE CURRENT SITUATION BRIEFLY AND SPREADING AWARENESS BY PROVIDING THE LATEST STATISTICS, NEWS, CARE GUIDELINES AND HELP RELATED TO IT, ALL IN ONE PLACE.

IN ADDITION TO THIS WE HAVE CREATED OUR OWN API SERVICE TO PROVIDE COVID-19 WORLD DATA THAT OTHER DEVELOPERS CAN USE IN THEIR OWN PROJECTS. WE HAVE ALSO DEVELOPED A BROWSER EXTENSION WHICH IS AVAILABLE IN THE CHROME WEB STORE THAT USES THE SAME API WHICH WE ARE PROVIDING AS A SERVICE.

THE STATISTICAL DATA IS SOURCED FROM WORLDOMETER'S AND JOHN HOPKIN'S REALTIME UPDATES, AND ALL THE INFORMATION RELATED TO CARE AND FAQs IS SOURCED FROM WORLD HEALTH ORGANISATION'S HELP AND SUPPORT.

WE HAVE WORKED TO PROVIDE THE BEST INFORMATION POSSIBLE WHICH YOU CAN TRUST AND FOLLOW CONFIDENTLY.

THE INFORMATION THAT YOU SEE UPDATES EVERY TEN MINUTES AS PER GMT STANDARDS, TO PROVIDE THE MOST RECENT UPDATES ON THE PANDEMIC.

WE WANTED PEOPLE TO SEE THIS AS SOMETHING THAT BRINGS US ALL TOGETHER. THIS IS NOT ABOUT ONE'S OWN COUNTRY; IT IS ONE PLANET - AND IT IS DISHEARTENING TO SEE OUR PLANET IN THIS STATE TODAY.

THIS PROJECT IS 100% NON-COMMERCIAL AND IS ONLY UNDERTAKEN TO GIVE BACK TO THE COMMUNITY BY HELPING THE SOCIETY.
